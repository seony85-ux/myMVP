# 1. 인트로 화면 (Intro)
목적: 서비스 분위기 전달 및 루틴 시작 진입점
- 상단: 앱명 또는 짧은 태그라인
- 중앙: 짧은 안내 문구 ("지금 이 순간, 나에게 집중해보세요")
- 하단: 주요 CTA 버튼 [루틴 시작하기]

# 2. 루틴 설정 화면 (Emotion + BGM + Skincare)
목적: 루틴 시작 전 주요 설정을 한 화면에서 완료하여 진입 속도 및 몰입도 향상

## 상단: 화면 제목 ("오늘의 루틴 설정")

## 섹션 1: 감정 선택
- 질문 텍스트 ("지금 기분은 어떤가요?")
- 감정 점수 선택 UI (1~5 단계, 이모지 또는 원형 버튼)

## 섹션 2: BGM 선택
- 섹션 타이틀 ("배경음")
- BGM 리스트 카드 (3개 + 없음)
- 간단한 분위기를 알 수 있는 BGM명
- 선택 시 하이라이트

## 섹션 3: 스킨케어 단계 선택
- 섹션 타이틀 ("스킨케어 단계")
- 단계 선택 토글
- 토너 / 에센스(또는 로션) / 크림 / 올인원 (최소 1개 이상 필수)

## 하단: [다음]

# 3. 음성 가이드 설정 및 루틴 시작 진입 화면
목적: 음성 가이드 사용 여부 선택 및 최종 루틴 시작 진입
- 상단: 질문 텍스트 ("음성 가이드를 들으시겠어요?")
- 중앙: On / Off 토글 버튼
- 하단: [루틴 시작]

# 4. 루틴 진행 화면 (Routine)
목적: 명상 + 스킨케어 루틴 자동 진행
- 상단: 현재 단계명 표시 (예: "토너 단계")
- 중앙: 현재 문구(명상 문장) 표시
- 단계 진행 상태 시각화 (간단한 바 또는 점)
- 하단: [중단하기] 버튼 (단일 버튼, 강조)

# 5. 감정 변화 기록 화면 (After Emotion)
목적: 루틴 후 감정 상태 기록
- 상단: 질문 텍스트 ("지금 기분은 어떤가요?")
- 중앙: 감정 점수 선택 UI (1~5)
- 하단: [결과 보기]

# 6. 결과 요약 화면 (Summary)
목적: 루틴 효과 요약 및 피드백 수집
- 상단: 요약 제목 ("오늘의 루틴 요약")
- 중앙: 감정 변화 요약 (전/후 비교)
- 만족도 점수 선택 (1~5)
- 재사용 의향 선택 (Yes/No)
- 하단: [완료]
